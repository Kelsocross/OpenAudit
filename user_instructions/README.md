## System Requirements

### Supported Browsers:
- Google Chrome (recommended)
- Mozilla Firefox
- Microsoft Edge
- Safari (Mac)

### File Formats Supported:
- **Invoice Upload:** Excel (.xlsx, .xls), CSV (.csv)
- **Contract Upload:** PDF, Excel (.xlsx), Word (.docx)

### Required Data Fields:
- Tracking Number
- Carrier
- Service Type
- Date (Shipment Date)
- Delivery Date
- Billed Amount
- Weight
- Zone

## Best Practices

### Data Management
1. Keep original invoice files backed up
2. Use consistent file naming conventions
3. Upload data regularly (weekly or monthly)
4. Export and save audit results

### Claim Submission
1. Review claims before submitting
2. Organize by carrier
3. Keep submission records
4. Follow up on claim status

### Performance Optimization
1. Process files in batches under 10,000 rows
2. Close unnecessary browser tabs
3. Use wired internet connection for large uploads
4. Clear browser cache regularly
